A little bit about my journey.

2016, August 6

First week in devCodeCamp went well. I've learned:

  • HTML & CSS
  • Responsive Design
  • Batch files
  • GitHub
  • Command line

Not only did I learn a lot, but I've created a three page website on my hobbies, learned some neat tips and tricks for my computer AND learned more about the different parts to a computer. Also, for the first time ever, I built my own gaming computer with a huge budget of $20,000. LOL

Creating a computer with a budget of $20,000 was actually pretty hard for me to do. I've never really thought about all the different parts needed to create a gaming computer; and why would I want to build a computer for that much when I could go buy a brand new car or save it for a down payment on a house. Maybe if I was a huge gamer....I do enjoy playing Overwatch, Diablo, WoW, Oblvion, Final Fantasy, etc...so...I guess you could say that I'm a mini gamer, but I still wouldn't buy a computer for $20,000.

Today I have been working on my portfolio. It's slowly coming together but everything great takes time to perfect, right? I'm happy with how far I got today and I'm hoping to get a lot more done tomorrow.
